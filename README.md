# mach

[mach]ine

making games at the speed of sound

## the idea

### one-liner

empower spreadsheet nerds to make games

### ramblings

- goal is NOT to make a game engine within Google Sheets / Excel, but rather "use spreadsheets as a programming language"
- holy grail is that, without touching C# or setting up a dev env and IDE, one is still able to make games as-if one knows C#
- use Unity as its the most "accessible" and cross platform-able (and i know Unity myself lol)
- the spreadsheet-er can also easily partner with other artists / engineers that knows Unity
- as much as possible re-use existing functionality of Unity, not making wrappers over every single Unity API
- likely would be making much use of dragging and dropping references around Unity
